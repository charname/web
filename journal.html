2025-07-25 — Immich Functional After Dataset Fix

Immich became fully operational after correcting dataset types in TrueNAS SCALE. The initial issue was assigning the application dataset as Generic instead of App. Recreating the datasets with proper types resolved startup issues.

Discovered Immich’s CLI-based recursive upload method via SSH. This approach automatically detects duplicates, skips already-processed files, and clearly reports failed uploads with full file paths. This significantly reduced rework and improved visibility during ingestion.

Marked the Immich setup as a successful trial. HTTPS, encryption, and advanced features remain unexplored for now.

2025-07-24 — TrueNAS Core to SCALE Migration

Upgraded from TrueNAS Core to SCALE to support native Docker workloads. The in-place upgrade path failed due to version constraints, requiring a clean SCALE install and configuration import.

Although more time-consuming than expected, the migration provided a cleaner foundation for containerized applications like Immich.
